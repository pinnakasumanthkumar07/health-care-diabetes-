# health-care-diabetes-
# 🏥 Diabetes Prediction Web App

## 📌 Overview

This project is a Machine Learning-based web application that predicts whether a person is likely to have diabetes based on health parameters.

## 🚀 Features

* User-friendly web interface
* Real-time prediction
* Machine Learning model integration
* Scaled input data using preprocessing

## 🛠 Tech Stack

* Python
* Flask
* Scikit-learn
* HTML/CSS

## 📂 Project Structure

* app.py → Main application
* model.pkl → Trained ML model
* scaler.pkl → Data preprocessing
* templates/ → Frontend UI

## ▶️ Run Locally

pip install -r requirements.txt
python app.py
